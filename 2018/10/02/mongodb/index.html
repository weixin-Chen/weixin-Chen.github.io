1.下载 http://dl.mongodb.org/dl/win32/x86_64
教程 http://www.cnblogs.com/shawWey/p/9052470.html

教程

http://www.cnblogs.com/sweetchildomine/p/6397136.html

2.创建几个文件夹具体如下：数据库路径（data目录）、日志路径（logs目录）和日志文件（mongo.log文件）

配置系统变量

  1. 创建配置文件mongo.conf，文件内容如下：
    1.#数据库路径
    2.dbpath=D:\MongoDB\data
    3.#日志输出文件路径
    4.logpath=D:\MongoDB\logs\mongo.log
    5.#错误日志采用追加模式
    6.logappend=true
    7.#启用日志文件，默认启用
    8.journal=true
    9.#这个选项可以过滤掉一些无用的日志信息，若需要调试使用请设置为false
    10.quiet=true
    11.#端口号 默认为27017
    12.port=27017
    管理员身份打开
    4.启动MongoDB
    mongod –config “D:\MongoDB\mongo.conf”

5.创建并启动MongoDB服务

mongod –config “D:\MongoDB\mongo.conf” –install –serviceName “MongoDB” （配置）
net start MongoDB （启动）

用户 –auth

卸载MongoDB服务 mongod.exe –remove –serviceName “MongoDB”

使用

数据库主要分成两种:

-关系型数据库(RDBMS)

  • MySQL,Oracle,DB2,SQL Server …

  • 关系数据库中全部都是表
    -非关系型数据科
    -MongoDB,Redis…
    -键值对数据库
    -文档数据库MongoDB

    sql : 结构化查询语言
    三个概念
    1数据库: database
    2 集合: collection
    3 文档: document

修改

db.collectionupdate(查询条件,新对象)

  • .update() 默认会把使用新对象替换成旧对象
    只会修改一个===db.collection.replaceOne()
    -如果需要修改指定的属性
    $set可以用来修改或添加文档中的指定属性
    db.stus.update({name:'孙悟空'},{$set:{age:28}})
    `$unset` 删除文档中的指定属性
    db.stus.update({name:'孙悟空'},{$unset:{age:1}}) 删除age

db.collection.updateMany() 修改多个

db.collection.updateOne() 一个

$push 向数组添加一个新的元素
db.collection.update({username:’xiaoming’,{$push:{‘hobby.movies’:’Interstellar’}}})

$addToSet 向数组中添加一个新的元素 (不重复的元素)
例子:
db.stus.update({age:10},{$addToSet:{like:'sport'}})
{
“_id” : ObjectId(“5b35b84d36c5fe2e4211d5f5”),
“age” : 10.0,
“sex” : “男”,
“isDel” : 0.0,
“like” : [
“sport” //添加的元素
]
}

删除

db.collection.remove(删除的条件) 跟find()一样,删除符合条件的文档(多个)
只删一个 db.collection.remove(删除条件,true)
db.collection.remove({}) //全删 性能差

db.collection.drop() 删除集合 没了数据,数据库就没有了
db.dropDatabase() 删除数据库
db.collection.deleteOne() 一个
db.collection.deleteMany() 多个

##不完全删

db.collection.updateMany({},{$set:{isDel:0}}) //所有添加isDel
db.collection.upadte({name:’ts’},{$set:{isDel:1}}) //假装删
db.collection.find({isDel:0}) //查看结果

比较和筛选

添加2000条数据

1到2000

  • $gt

db.numbers.find({num:{$gte:500}}) num>=500的

  • $eq 等于

  • $lt 小于 -$or:[ ] -$inc 自增

例子:

db.numbers.find({num:{$gt:40,$lt:50}}) 40<num<50 并

db.numbers.find({$or:[{num:{$lt:10}},{num:{$gt:1990}}]}) num<10 或="" num="">1990

db.numbers.find({num:{$lte:10}}) 1-10 10条数据

db.numbers.find().limit(10) 前10条数据

db.numbers.updateMany({},{$set:{val:10}}) 所有插入 val:10

db.numbers.updateMany({num:{$lte:10}},{$inc:{val:+10}}) num< 10 val+10

分页

每页显示10条 跳过 页码
1-10 0 1
11-20 10 2
21-30 20 3

….
公式
.skip((页码-1)*每页显示的条数).limt(每页显示的条数)

skip() 跳过指定数量的数据
mongodb 会自动调整limt和skip的位置

db.numbers.find().skip(10).limit(10) 11条到20条

排列

sort

db.collection.find({})

查询文档时,默认情况是按照_id的值进行排序(升序)
sort() 可以用来指定文档的排序的规则
sort() 中需要传递一个对象来指定排序的规则

db.collection.find({}).sort({val:1}) 1 表示升序 -1 降序

limit skip sort 可以任意的顺序进行调用

sort>skip>limit

在查询时,可以在第二个参数的位置来设置查询结果的 投影(想要显示的字段)
db.numbers.find({},{val:1,_id:0})
{
“val” : 20.0
}
